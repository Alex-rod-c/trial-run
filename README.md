# ThreadLab storefront

This repository contains a single-page storefront concept for "ThreadLab", an
apparel brand designed to feel playful while staying polished. The experience
includes:

- A hero section that highlights the latest collection and brand commitments.
- Dynamic deal tiles and a live banner that spotlight stackable savings.
- A shoppable product grid with category tabs, sorting, ratings, and pricing.
- A persistent cart drawer with quantity controls, coupon handling, automatic
  shipping perks, and localStorage persistence.
- Light/dark theme toggling and responsive layout refinements for small
  screens.

To preview the site locally, start any static file server from the repository
root (for example, `python -m http.server 8000`) and open `http://localhost:8000`
in your browser.
